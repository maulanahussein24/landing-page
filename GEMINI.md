# Project: Landing Page with Product Management

## Directory Overview
Ini adalah project landing page yang lagi dikembangin, pakai **Astro** sebagai framework utamanya. Tujuan utamanya adalah punya sistem login biar lu bisa gampang nge-upload dan ngelola produk-produk baru tanpa harus ngoding manual lagi.

Awalnya, di sini ada `README.md` sama beberapa file gambar (`Desktop - X.png`) yang kemungkinan besar jadi aset visual atau mockup desain. Sekarang, fokusnya nambahin fitur login dan pengelolaan produk.

## Key Files
*   `README.md`: Deskripsi awal project sebagai "Landing Page".
*   `Desktop - 1.png` sampai `Desktop - 6.png`: File-file gambar ini bisa jadi mockup desain, screenshot, atau aset visual buat landing page lu.

## Usage
Project ini sekarang punya roadmap ke arah:
*   **Frontend:** Dibangun pakai Astro.
*   **Login System:** Akan ada sistem login untuk admin atau user tertentu.
*   **Product Management:** Setelah login, user bisa upload dan ngelola produk baru dengan mudah.

Kedepannya, lu bakal nambahin:
*   Komponen Astro (`.astro` files) buat halaman-halaman website.
*   Logika backend atau integrasi API buat sistem login dan manajemen produk.
*   CSS buat styling yang lebih rapi dan modern.
*   JavaScript buat interaktivitas, kalau perlu.

Aset gambar yang udah ada bisa langsung dipakai atau diatur ulang sesuai kebutuhan desain.
